Kickstarter is a cultural phenomenon

It has changed the world. People who never before had access to funding for their creative projects, ideas that never would have made it past day dreams, even some things that shouldn't have seen the light of day all defied what previous generations thought was possible. Kickstarter is, one of many, cases and points for transformative power of the internet. It demonstrates the possbility that commerce and creativity and fun and play and work can all happen at the same time.

Kickstarter inspires us. So we made some tools and gathered some data to help the community continue to surprise eachother.

The Dataset

We extracted key metrics from over 30'000 projects featured on the Kickstarter website. The above charts are a just very small sampling of what insights are possible to accrue from this wealth of information. Here is a brief list of the information we extracted from each of the 30'000+ projects we gathered:

  • Project Title
  • Project Creator
  • Project Category
  • Project Location
  • Project Start Date
  • Project End Date
  • Data Set Last Updated Date
  • Total Amount Pledged to Project
  • Project Funding Goal
  • Percent Raised of Funding Goal
  • Project Currency
  • Project Status (e.g. live, successful, etc)
  • Project Backers (names & locations of backers that made their information publically available)
  • The Different Pledge levels per project (including number of backers per pledge level, number of pledge levels per project)

To grab a copy of the data yourself, available in both JSON format and the raw HTML source data, visit us on github. Also on github, we have the software that we used to gather this data, which you can use to further update or expand the dataset yourself.
